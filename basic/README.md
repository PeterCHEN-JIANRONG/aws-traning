# AWS 基礎課程

AWS 教育訓練

## 介紹

此次開課課程為 `AWS 基礎課程` 將會包含以下項目：

- AWS 概略介紹
- LAB 實戰：
  - VPC (Subnet, IGW, NAT Gateway, Route Table)
  - EC2 (Instances, AMI, Security Groups, Load Balancers, AutoScaling)
  - S3
  - CloudWatch
  - CI / CD (Github, Travis CI, CodeDeploy)
  - Tools (packer)
